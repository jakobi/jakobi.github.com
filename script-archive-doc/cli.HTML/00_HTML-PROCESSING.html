Peter's Homepage - HTML

Commands dealing with HTML. Most of these are pretty ancient, but sometimes still surprisingly useful.

Major Commands (early 90's style)

  • COLLECT (turns a set of html files into a single html document)
  • DIFF, DIFFTREE (from single html documents to directories)
  • htmlcheck and files (a truly ancient html validator; I think the configuration and some suppression of warnings was all I changed)
  • LINKS (extracts the links from an HTML document)
  • XLINKS (link extraction and helper for spidering)
  • db_cgi.p is the retired DBFW's grab-bag of utility functions. Some scripts may still depend on them.
  • dir2html.pl old style configurable dirtree filelist using .name and .skip files for control. dir2array is a more recent variant: just create a perl hash using the same principles as dir2html.pl, and dump that to stdout for use in an embperl template (as a buzz-phrase: that's generating a static website using the MVC approach). Example code incl. the github site creation (exportscript) is in a subdir in this section as well.
