Peter's Homepage - Miscellaneous Scripts

Some scripts, helpers and experiments that don't quite fit the other categories.

Commands

  • ALARM (acoustic environmental damage, for use with waitcond)
  • df.minfree (checks free space and warns if lower than requested)
  • linkname (symlink to canonical name; when things like cd -P are unavailble)
  • mt_dir (configures directory short names and invokes one of echo/cd/mc; see also the nico_cd stack in the shell function section)
  • paradj (modified; can now skip \ to format e.g. double quoted text in perl)
  • repeat (when I'm too lazy to write a while ... do; command; done loop)
  • sanitize_filenames (rename files containing certain characters, umlauts, etc; see also myrename; superseeded by myrename)
  • sleep.p (select-based sub-second sleep)
  • sleepuntil TIMEEXPR [COMMAND [TIMEEXPR [COMMAND] ...]] (TIMEEPR is a either a command to run and wait for, or a string giving the delay time)
  • unespace.pl (modified version of Guide de Rosa's unespace; strip some of the noise from script's typescript sessions logs; tries to strip e.g. vi sessions from the logs to increase their usability)
  • watchfloat (write background command output to a floating region of a tty)
